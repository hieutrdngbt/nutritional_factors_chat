---
description: Generate or update API documentation
---

Generate comprehensive API documentation:

1. Analyze backend API routes and endpoints
2. Document each endpoint with:
   - HTTP method and path
   - Request parameters (query, body, headers)
   - Request body schema
   - Response format and status codes
   - Authentication requirements
   - Example requests and responses
   - Error responses

3. Organize by resource/domain
4. Include authentication/authorization details
5. Document rate limiting if applicable

Generate documentation in a clear, readable format (Markdown or OpenAPI/Swagger format).
